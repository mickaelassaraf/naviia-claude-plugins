# Naviia — plugins Claude

Marketplace de plugins Claude (Cowork / Claude Code / Desktop) de Naviia.

## Installation

```
/plugin marketplace add mickaelassaraf/naviia-claude-plugins
/plugin install naviia-two-pager@naviia
```

## Plugins

- **naviia-two-pager** — profil d'entreprise PE complet : classification du
  marché, recherche web, sections rédigées en sous-agents à contexte isolé,
  rendu PDF identique au portail Naviia. Les prompts métier et le rendu sont
  servis par le connecteur MCP Naviia (monté automatiquement par le plugin) ;
  la génération tourne sur l'abonnement Claude de l'utilisateur.
